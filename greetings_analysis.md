# 인사말 서브페이지 레퍼런스 분석 & 설계

| 사이트 | 서브 히어로 | 브레드크럼 | 좌측 LNB | 본문 구성 | 인물 사진/서명 |

## 2. 공통 IA (우리 사이트 = ANCHOR · 충남형 RISE)
- 우리 메인 GNB 1Depth: 센터소개 / 인재양성 / 연구개발 / 지역사회혁신 / 사업·성과관리 / 알림마당 / 홍보마당
- 인사말은 "센터소개" 하위.
- 센터소개 2Depth(기존 IA): 인사말 / 비전 및 전략 / 조직도 / 찾아오시는 길

## 3. 서브페이지 공통 레이아웃 설계 (확정 메인 디자인 언어 계승)
메인 시안 특징: 곡선 웨이브 디바이더, #003D92 Royal Blue + Teal + Amber, Pretendard+Space Grotesk, 풀스크린 히어로.

### 공통 컴포넌트
1. **SubHero (서브 히어로 배너)**
   - 높이 ~280px, 블루 그라데이션 배경 + 배경 이미지 오버레이 + 하단 곡선 웨이브(메인과 동일 모티프)
   - 페이지 타이틀(국문 대형 + 영문 eyebrow)
   - 우측 정렬 브레드크럼 (HOME > 센터소개 > 인사말)
2. **SubLayout (2단 레이아웃)**
   - 좌측: LNB(현재 섹션 타이틀 헤더 + 2Depth 메뉴 리스트, 현재 active 표시, 화살표)
   - 우측: 본문 콘텐츠 영역
   - 모바일: LNB는 상단 가로 스크롤 탭 또는 드롭다운으로 축소
3. **Breadcrumb**: SubHero 내부 + 모바일 대응

### 인사말 콘텐츠 영역 구성
- 좌측: 사업단장 사진(AI 생성, 전문적) + 네임카드(직함 + 성명) — 서울과기대 네임카드 스타일 참고
- 우측: "안녕하십니까?" 대형 강조 헤드라인 + 본문 다단락 + 핵심 키워드 하이라이트
- 하단: 4대 비전 키워드 칩 또는 핵심 가치 카드 (UX 고도화 — 단순 텍스트 나열 회피)
- 서명 영역: "충남대학교 RISE 사업단장 OOO" + 친필 느낌 서명 이미지(선택)
- CTA: "비전 및 전략 보기" / "조직도 보기" 연계 링크 (회유 동선)
